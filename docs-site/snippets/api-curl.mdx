All API examples use `http://127.0.0.1:4000` as the base URL. For remote access, replace with your server address and add the bearer token:

```bash
curl -s http://127.0.0.1:4000/api/v1/state \
  -H 'Authorization: Bearer $RISOLUTO_WRITE_TOKEN'
```

See the [Network Security guide](/guides/security) for write token and bind address configuration.
